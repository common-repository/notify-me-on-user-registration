A secondary notification to receive when a new user registers

== Description ==

This plugin can be used to add an email address in addition to existing admin email and will receive an extra email notification whenever a new user is registered. You can install it just by downloading. Its very simple and easy.


== Frequently Asked Questions ==

= Will this plugin damage my website? =

This plugin is free and completely safe.

= Where are the menus for adding email address? =

You can find "Notify Me" tab in the left menu of wordpress admin panel where you can input the email address or even select the functionality of this plugin.
